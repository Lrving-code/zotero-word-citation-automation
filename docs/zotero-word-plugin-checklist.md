# Zotero Word plugin checklist

Use this checklist before reporting a workflow bug:

1. Zotero Desktop is openable on this machine
2. Microsoft Word opens documents normally
3. The `Zotero` ribbon is visible in Word
4. `Document Preferences` responds
5. `Refresh` works in a normal Zotero-managed document
6. The generated `.docx` opens without repair prompts

If step 3-5 fail, fix the Zotero Word integration first. The package assumes the plugin is already functional.
