# Release v0.1.0

Initial public release of `zotero-wordflow`.

## Highlights

- Turns DOI-backed prose and reference lists into Zotero-ready Word documents.
- Verifies references through Crossref before Zotero import.
- Writes `ADDIN ZOTERO_ITEM CSL_CITATION` fields directly into `.docx`.
- Keeps the final bibliography generation inside Microsoft Word through the Zotero add-in.
- Includes a Codex skill adapter for repeatable cross-thread use on Windows.

## Included in this release

- Python package with CLI:
  - `zotero-wordflow build-manifest`
  - `zotero-wordflow run`
- Example prose, references, and minimal manifest.
- Basic CI workflow and pytest coverage.
- Installation helpers and Windows setup documentation.

## Validation status

- Editable install verified locally.
- `pytest` passed locally.
- Manifest generation smoke check passed locally.

## Known limitations

- Parenthetical author-year citations are supported.
- Narrative citations such as `Smith (2020)` are not auto-converted yet.
- This release targets local Zotero Desktop plus Microsoft Word on Windows.
- Final bibliography insertion still happens through the Zotero Word add-in.

## Recommended next steps

- Add narrative citation support.
- Add dry-run planning mode for Zotero import.
- Expand mixed-language reference parsing coverage.
